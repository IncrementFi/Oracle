# Increment Oracle


### Deployment:
#### Emulator
1. Run `flow emulator -v` to start emulator
2. Run `python ./scripts/deployment/emulator/deploy.sh` to deploy accounts and contracts.
3. Run `python ./scripts/deployment/emulator/test.sh` for testing.
